# 🍽️ Restaurant Billing & Menu Management System

A desktop-based **Restaurant Billing and Menu Management System** developed to simplify restaurant order processing, billing, and receipt generation. The application provides an easy-to-use graphical interface for selecting food and beverages, calculating bills automatically, and generating customer receipts.

## 📖 Overview

The Restaurant Billing System is designed to automate the billing process in restaurants, cafes, and food courts. Users can select menu items, specify quantities, calculate the total amount, and generate a formatted bill within seconds.

The system minimizes manual calculation errors while providing a simple and efficient billing experience.

## 🚀 How It Works

1. Launch the application.
2. Enter the quantity for each food or drink item.
3. Click **SUM** to calculate the total.
4. Click **BILL** to generate the customer receipt.
5. Click **NEW BILL** to clear all fields and start a new order.
6. Click **CLOSE** to exit the application.
